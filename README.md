# 🥷 Kakashi's Quest — 2D Ninja Platformer

A Naruto-themed 2D browser platformer where you play as Kakashi Hatake,
fighting through Sound Ninja enemies using combat abilities like
Chidori and Sharingan.

## 🎮 Gameplay Features
- Play as Kakashi with punch, kick, Chidori & Sharingan abilities
- Enemy AI with patrol, detection and attack behavior
- Health system with invulnerability frames
- Victory and Game Over screens
- Pixel art rendering

## 🛠️ Built With
- [Phaser 3](https://phaser.io/) — 2D game framework
- [Vite](https://vitejs.dev/) — build tool and dev server
- JavaScript (ES6 modules)

## 🚀 How to Run Locally
Make sure you have [Node.js](https://nodejs.org/) installed, then:

```bash
npm install
npm run dev
```
Then open your browser at `http://localhost:5173`

## 📦 Build for Production
```bash
npm run build
```
